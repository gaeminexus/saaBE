# G41 — ParticipeActivoG41

> ⚠️ Si se modifica la lógica de este reporte, actualizar este archivo.

**Tabla destino:** `RPR.CG41`
**Servicio:** `GeneracionG41ServiceImpl.java`
**Fuente:** `CRD.ENTD` con `idEstado = 1` (nuevas, no procesadas)

## Lógica
1. Obtener todas las entidades con `idEstado = 1`. Si no hay → retornar 0, EJRD OK.
2. Por cada entidad: buscar su `Participe` (por `entidad.codigo`) y su `Exter` (por `entidad.numeroIdentificacion`).
3. Mapear a CG41.
4. Actualizar `entidad.idEstado = 10` (procesada).
