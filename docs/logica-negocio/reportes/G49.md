# G49 — CancelacionG49

> ⚠️ Si se modifica la lógica de este reporte, actualizar este archivo.

**Tabla destino:** `RPR.CG49`
**Servicio:** `GeneracionG49ServiceImpl.java`
**Fuente:** `CRD.DTPR` (DetallePrestamo) — tres grupos

## Grupos

**Grupo 1:** Préstamos cuya cuota con **mayor numeroCuota** esté dentro del mes y sea `estado=4` (pagada). Indica cancelación normal.

**Grupo 2:** Préstamos con `estadoPrestamo=4` (cancelado anticipado) cuya cuota con **mayor numeroCuota pagada** esté dentro del mes. Excluye los del Grupo 1.

**Grupo 3:** Operaciones que estaban en el G48 del período anterior pero **ya no aparecen** en el G48 actual. Si no hay EJRC anterior → grupo 3 vacío.

Se usa un `Set<numeroOperacion>` para evitar duplicados entre los tres grupos.

## Campos

| Campo | Valor |
|---|---|
| `tipoIdentificacion` | **FIJO: `"C"`** |
| `identificacion` | `Prestamo.entidad.numeroIdentificacion` (grupos 1 y 2) / `CG48anterior.identificacion` (grupo 3) |
| `numeroOperacion` | `Prestamo.idAsoprep` como String |
| `fechaCancelacion` | Último día del mes de ejecución |
| `formaCancelacion` | **FIJO: `"N"`** |
