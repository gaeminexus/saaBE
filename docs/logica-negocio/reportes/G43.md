# G43 — ParticipeCesanteG43

> ⚠️ Si se modifica la lógica de este reporte, actualizar este archivo.

**Tabla destino:** `RPR.CG43`
**Servicio:** `GeneracionG43ServiceImpl.java`

## Lógica
1. Obtener identificaciones del G42 del mes actual → conjunto "activos".
2. Obtener identificaciones del G42 del mes anterior:
   - Si existe EJRC anterior → usar `RPR.CG42` de ese EJRC.
   - Si NO existe EJRC anterior → usar `RPR.HM42` (HistoricoG42).
3. `cesantes = universo_anterior - activos_actuales`
4. Insertar en CG43 cada cesante con `identificacion`, `tipoIdentificacion` y `detalleEjecucion`.
5. Si no hay cesantes → retornar 0, EJRD OK.

## Campos
Los campos de liquidación (`fechaTerminoRelacionLaboral`, `fechaLiquidacion`, `saldoCuentaIndividual`, `valoresCompensados`, `valoresPagados`, `numeroImposicionesPersonales`, `numeroImposicionesPatronales`) quedan en `null` — provienen de procesos operativos externos.
