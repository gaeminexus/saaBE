# G47 — NovacionG47

> ⚠️ Si se modifica la lógica de este reporte, actualizar este archivo.

**Tabla destino:** `RPR.CG47`
**Servicio:** `GeneracionG47ServiceImpl.java`
**Fuente:** `CRD.PRST` con `estadoPrestamo = 9` (novaciones)

## Lógica
1. Obtener todos los préstamos con `estadoPrestamo = 9`. Si no hay → retornar 0, EJRD OK.
2. Por cada préstamo → mapear a CG47.

## Campos

| Campo | Valor |
|---|---|
| `tipoIdentificacion` | **FIJO: `"C"`** |
| `identificacion` | `Prestamo.entidad.numeroIdentificacion` |
| `numeroOperacion` | `Prestamo.idAsoprep` como String |
| `fechaNovacion` | `Prestamo.fecha.toLocalDate()` |
| `numeroOperacionAnterior` | `null` (no disponible) |
